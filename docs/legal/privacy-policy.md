# プライバシーポリシー

**最終更新日:** 2026-05-23
**サービス名:** Oikomi（追い込み）
**提供者:** Shu Hirouchi（以下「当方」）

本ポリシーは、当方が提供する筋トレ記録アプリ「Oikomi」（以下「本アプリ」）における個人情報および利用者データの取り扱い方針を定めるものです。

---

## 1. 概要

本アプリは、利用者の Apple デバイス（iPhone / Apple Watch / Mac / iPad）上で筋力トレーニングの記録と分析を行うアプリです。当方は外部サーバーを運用せず、利用者のデータは原則として **利用者自身のデバイスおよび Apple の iCloud 内** にのみ保存されます。第三者の分析 SDK、広告ライブラリ、トラッキングツールは一切使用していません。

---

## 2. 収集するデータ

### 2.1 HealthKit から読み取るデータ（任意）

利用者の許可がある場合に限り、Apple HealthKit から以下を読み取ります。読み取り対象は Pro プラン契約者のみとなる項目を含みます。

- 心拍変動 (HRV, SDNN)
- 安静時心拍数
- 睡眠データ（睡眠分析カテゴリ）
- 体重
- 体脂肪率
- 除脂肪体重

読み取ったデータは **端末内のみ** で処理し、外部送信は一切行いません。

### 2.2 HealthKit へ書き込むデータ（任意）

完了したワークアウトを `HKWorkout`（種別: トラディショナル筋力トレーニング）として「ヘルスケア」アプリに保存します。これにより Apple Watch のアクティビティリングに反映されます。書き込みは利用者が許可した場合のみ行われます。

### 2.3 アプリ内で記録するトレーニングデータ

利用者が本アプリ内で入力・記録する以下のデータを端末内に保存します。

- ワークアウトセッション（開始日時 / 終了日時 / メモ）
- セット記録（重量 / レップ数 / 所要秒数 / RPE / 推定 1RM / レスト秒数）
- 種目情報（カスタム種目を含む種目名・部位・器具・場所）
- ルーティン情報（名前 / 計画セット / 計画レップ / 計画重量）
- 自己ベスト（種目別の最大挙上重量 / 推定 1RM / 達成日）
- HealthKit から取得したスナップショット（セッション単位）

### 2.4 設定値

以下を端末内 `UserDefaults` および共有 App Group コンテナに保存します。

- iCloud 同期の有効化フラグ
- 週次トレーニング目標日数
- アプリアイコン選択状態
- オンボーディング完了フラグ
- 種目ライブラリのシード初期化フラグ
- Pro サブスクリプション有効状態のキャッシュ（権利の真実のソースは Apple App Store）

### 2.5 サブスクリプション情報

Pro プランは Apple の StoreKit 2 を通じて販売され、購入情報・トランザクション情報は **すべて Apple App Store が管理** します。当方はこれらの情報を独自に保存・送信していません。

### 2.6 取得しない情報

本アプリは以下を **取得しません**:

- メールアドレス、電話番号、氏名
- 位置情報、写真、連絡先、カレンダー、マイク、カメラ
- IP アドレス、デバイス識別子による行動追跡
- 第三者サービスへの送信用識別子

## 3. データの保存場所

### 3.1 端末内（SwiftData / App Group）

トレーニング記録および設定値は、利用者の Apple デバイス内の SwiftData ストアと App Group 共有コンテナ（`group.com.shuhirouchi.oikomi`）に保存されます。このコンテナは iPhone / Apple Watch / Mac / ウィジェット拡張機能の間で共有され、いずれも利用者の端末内に閉じています。

### 3.2 iCloud（Pro プラン限定・任意）

Pro プラン契約者は、設定画面から iCloud 同期を有効化できます。有効化すると、上記 §2.3 のデータが Apple の CloudKit コンテナ（`iCloud.com.shuhirouchi.oikomi`）に保存され、利用者の Apple ID に紐づく複数デバイス間で同期されます。

- CloudKit は **オプトイン** で、設定画面からいつでも無効化できます
- CloudKit のデータは **利用者の Apple ID で自動隔離** され、当方は内容を閲覧できません
- 無料プランでは iCloud 同期は行われず、データは端末内のみに保存されます

### 3.3 HealthKit（Apple 管理）

§2.1 / §2.2 のデータは Apple の HealthKit が管理しており、利用者の許可した範囲でのみ本アプリが読み書きします。HealthKit データの保存・削除は「ヘルスケア」アプリで行ってください。

### 3.4 Apple Watch ↔ iPhone 同期

Apple Watch と iPhone の間では `WatchConnectivity` フレームワーク（WCSession）を通じてセッションやレストタイマーの状態がリアルタイムに同期されます。この通信は Apple のシステム内で完結し、外部サーバーを経由しません。

## 4. データの第三者送信について

本アプリは以下を遵守しています:

- **第三者分析 SDK は一切使用しません**（Firebase / Sentry / Mixpanel / Amplitude などを含む、すべて不採用）
- **広告ライブラリは一切使用しません**
- **トラッキング技術は一切使用しません**
- **外部 API へのデータ送信は行いません**（CloudKit / HealthKit を除く、すべて Apple のフレームワーク経由）

本アプリの通信は Apple のシステムサービス（App Store / HealthKit / CloudKit / WatchConnectivity / iCloud）に限定されます。

---

## 5. 利用者の権利

利用者は以下の操作をいつでも行えます。

- **データの削除**: アプリ内の設定画面から、すべてのトレーニング記録を削除できます
- **iCloud 同期の停止**: 設定画面のトグルから iCloud 同期を無効化できます
- **HealthKit 連携の解除**: iOS「設定」→「ヘルスケア」→「データのアクセスとデバイス」から本アプリのアクセス権を取り消せます
- **データのエクスポート**: Pro プランでは CSV 形式でトレーニングデータをエクスポートできます
- **サブスクリプションの解約**: iOS「設定」→「自分の名前」→「サブスクリプション」から行えます

EU 圏（GDPR）および日本の個人情報保護法に基づくアクセス権・訂正権・削除権の行使を希望される場合は、§9 の連絡先までお問い合わせください。

## 7. データの保管期間

本アプリで記録されたデータは、利用者がアプリ内で削除するか、デバイスからアプリをアンインストールするまで保持されます。アプリをアンインストールした場合、端末内のデータは削除されますが、iCloud 同期を有効にしていた場合は CloudKit 上のデータが残ります（他デバイスで再ログインすると復元可能）。CloudKit 上のデータを完全削除するには、iOS「設定」→「自分の名前」→「iCloud」→「ストレージを管理」から本アプリのデータを削除してください。
